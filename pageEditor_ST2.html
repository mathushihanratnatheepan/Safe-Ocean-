Home Page

Technical Description

The header tag is used for the navigation bar and the logo. There are multiple section tags to organize content like home,marine biodiversity, content, importance of marine diversity, ready to volunteer adopting footer.Internal links like #Important are there to jump within the web content page.There are external links to social media via icons using boxicons. Images are used to illustrate the marine life topics that connect with our website using the img tag. The links to external stylesheets like style.css which is the main css is linked and others like the boxicons, remixicon and google fonts are used for icons and modern fonts to be used.

Accessibility

The accessibility of the home page focuses on providing an inclusive experienced for all users, regardless of their abilities. It includes features such as a keyword navigation to navigate them to the pages they want to refer. It also includes screen reader compatibility through proper labeling and alt text and sufficient color contrast for better readability. The design ensures that the content is easy to understand and navigate, making the website accessible to people with visual,audiority or motor impairments.

Table Page

Technical Description

This page is structured using semantic HTML elements for clarity and accessibility. The main content is wrapped within a main tag , which designates the primary focus of the page. Inside it there are two sections which is one for the table header containing the title and another for the table body that holds the actual data table. The table uses standard table , thead , tbody , and tr elements to organize data in rows and columns. Each th tag contains both text and icons. The data in the table shows an analysis on the pollution of water bodies over the years. The styling is done by their names and each of them has unique colors and hover effects that suit them.

Accessibility

The accessibility to this page user friendly and very interactive with the users without the age difference. To access this page from navigation bar the user could click the table statement it will navigate to this page. The colors which are used connects to the oceans and all the table headers has an icon and a related text to it which makes more understanable. This page uses a external stylesheet called the table.css which makes it more organized. Overall this page is designed to be both informative and accessible, with clear separation of content and style.

Challenges and Lessons Learned

One key challenge was creating responsive, accessible designs while managing layput and styling issues. Debugging and ensuring cross-browser compatibility also was difficult.

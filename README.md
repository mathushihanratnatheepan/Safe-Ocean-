# Ocean Conservation Website

A collaborative web design project focused on marine biodiversity and ocean conservation awareness.

## Project Overview

This website educates users about the importance of ocean conservation, marine life protection, and environmental sustainability. It covers topics including plastic pollution, coral reef preservation, and future ocean challenges.

## Features

- **Responsive Design** - Works on desktop, tablet, and mobile devices
- **Interactive Navigation** - Fixed header with smooth scrolling
- **Content Sections** - Multiple pages covering ocean conservation topics
- **Volunteer Portal** - Information about volunteering opportunities
- **Contact Form** - Feedback and inquiry submission
- **Team Page** - Information about project contributors

Team Page

Technical Description

The Team page showcases the project group members using a responsive card-based layout built with CSS Grid. Each team member card displays their name, role, student ID, and pages developed. The page uses CSS custom properties (CSS variables) for consistent theming. Hover effects are implemented using CSS transitions and transforms for interactive feedback. The card design includes nested grid layouts for the "Pages Developed" section with anchor links to each student's work. Emoji characters are used as avatars for a friendly, accessible design without requiring external image assets.

Accessibility

The Team page is designed with comprehensive accessibility features. The card layout uses semantic HTML structure with proper heading hierarchy. All links have descriptive text indicating their destination. The hover and focus states provide clear visual feedback for interactive elements. The responsive design uses CSS Grid auto-fit to adapt from 4 columns to 2 to 1 based on viewport width, ensuring readability on all devices. Color contrast ratios meet WCAG 2.1 AA standards. The group information section uses clear labels and large, readable text.

Sitemap Page

Technical Description

The Sitemap page provides a comprehensive overview of the website structure using multiple visualization methods. The main content uses a CSS Grid layout with six category sections, each containing semantically structured lists of page links. Each link includes a description using paragraph elements for context. A visual tree diagram is implemented using CSS Flexbox and pseudo-elements to show the site hierarchy. The page uses internal CSS for complex layout-specific styles while maintaining consistency with the global stylesheet.

Accessibility

The Sitemap page prioritizes navigation accessibility with clearly organized categories and descriptive link text. Each category uses heading elements (h2) for easy screen reader navigation. The visual tree diagram is supplementary, with all links also available in the text-based sitemap above. Links have clear hover and focus states with transform effects for visual feedback. The page descriptions provide context for each link destination. The responsive design reorganizes content for optimal viewing on all screen sizes without losing functionality.

Content Page - Impact of Plastic Pollution

Technical Description

The Plastic Pollution content page presents critical information about ocean plastic pollution using the established content page template. The hero section features a full-viewport background with gradient overlay and impactful statistics. Content sections alternate between 'marine' and 'Protect' layouts using CSS Grid for image-text arrangements. Internal anchor navigation (#microplastics) allows users to jump to specific sections. The page integrates with other content pages through cross-links, creating a cohesive learning experience about ocean conservation topics.

Accessibility

This content page maintains strong accessibility standards throughout. All images have descriptive alt text explaining their content (e.g., "Sea Turtle", "Ghost Fishing"). The heading structure follows proper hierarchy from h1 through h5 for logical content organization. Bold text is used for important statistics and facts without relying on color alone for emphasis. Internal links use clear, action-oriented text. The responsive design ensures images scale appropriately and text remains readable on all devices. The "Go to top" navigation aids users with mobility impairments or those using keyboard navigation.
